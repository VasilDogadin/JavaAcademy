Задание №1 - Список спец номеров.
1. Создать класс машина. У машины есть номер.
2. Создать 50 машин с номерами а0[01-50]ан799 (где [01-50] - это все значения от 01 до 50)
3. Создать 50 машин с номерами к0[01-50]се178
4. Соединить машины в единый стрим
5. Машины, с номерами 04[0-9] - это спец номера (врачи) оставить только те машины, которые обладают спец номерами
6. Получить из машин номера
7. Соединить в единую строку, с разделителем в " ," - самостоятельное изучение ожидаемый финал
"a040ан1799, a041ан799, ... , k048се178, k049се178"


Задание №2 - Сколько здесь "о" ?
1. Создать набор уникальных слов: "тонь", "тополь", "боль", "рой", "стройка"
2. Внутри одного стрима:
2.1 посчитать количество букв "о" в одном слове,
2.2 посчитать сумму количества букв "о" во всех словах из набора,
3. Если буквы "о" есть - напечатать количество букв "о", если нет выкинуть исключение ожидаемый результат: 6


Задание №3 - Небоскребы, небоскребы, а я...
1. Дано: Небоскреб - имя небоскреба, его высота в метрах.
2. Необходимо создать небоскребы:
        2.1 Всемирный торговый центр 1 - 541м
        2.2 Шанхайская башня - 632м
        2.3 Бурдж-Халифа - 828м
        2.4 Международный финансовый центр Пинань - 599м
        2.5 Абрадж аль-Бейт - 601м
        2.6 Всемирный центр Лотте - 555м
3. Занести данный список в List! Дважды положить бурдж халифа в лист.
4. С помощью стримов:
        4.1 Убрать дубликаты (РАЗНЫЕ СТРИМЫ)
        4.2 Вывести только первые три небоскреба
        4.3 Вывести самый высокий небоскреб
        4.4 Вывести те небоскребы, которые выше километра
        4.5 Если выше нет, то вывести на экран: небоскреба выше километра - нет
